## Технологии

### Бэкенд:
- .NET 9.0
- Entity Framework Core
- PostgreSQL
- Swagger/OpenAPI

### Фронтенд:
- React 18
- Tailwind CSS
- Axios
- React Router

### Инфраструктура:
- Docker
- Docker Compose
- Nginx

## Функционал

- Просмотр новостей на русском и английском языках
- Админ-панель для управления новостями
- Загрузка новостей по URL
- Пагинация и фильтрация новостей
- Аутентификация администратора

## Запуск проекта

### Локальная разработка:

```bash
# Бэкенд
cd NewsPaper
dotnet run

# Фронтенд
